## 개요

C언어는 1972년 Dennis Ritchie가 개발한 절차적 프로그래밍 언어로, 임베디드 시스템에서 여전히 가장 널리 쓰입니다. Raspberry Pi, Arduino, STM32 등 MCU 펌웨어의 대부분이 C로 작성됩니다.

---

## 기본 데이터 타입

| 타입 | 크기 | 범위 | 용도 |
|---|---|---|---|
| `char` | 1 byte | -128 ~ 127 | 문자, 작은 정수 |
| `int` | 4 byte | -2³¹ ~ 2³¹-1 | 일반 정수 |
| `float` | 4 byte | ±3.4e38 | 단정밀도 실수 |
| `double` | 8 byte | ±1.7e308 | 배정밀도 실수 |
| `unsigned int` | 4 byte | 0 ~ 2³²-1 | 양의 정수 (센서값 등) |

임베디드에서는 크기가 명확한 `stdint.h` 타입을 선호합니다:

```c
#include <stdint.h>

uint8_t  sensor_val = 255;   // 0~255
int16_t  temperature = -10;  // 부호 있는 16비트
uint32_t timestamp = 0;      // 32비트 타이머 카운터
```

## 변수 선언과 초기화

```c
#include <stdio.h>

int main(void) {
    int count = 0;
    float voltage = 3.14f;  // f 접미사: float 리터럴
    char status = 'A';
    
    // 여러 변수 동시 선언
    int x = 0, y = 0, z = 0;
    
    printf("count: %d\n", count);
    printf("voltage: %.2f\n", voltage);
    printf("status: %c\n", status);
    
    return 0;
}
```

> 임베디드에서는 전역변수를 최소화합니다. 스택 오버플로 방지를 위해 지역변수 크기도 주의해야 합니다.

## 연산자

### 산술 연산자

```c
int a = 10, b = 3;
printf("%d\n", a + b);   // 13
printf("%d\n", a - b);   // 7
printf("%d\n", a * b);   // 30
printf("%d\n", a / b);   // 3  (정수 나눗셈, 소수점 버림)
printf("%d\n", a % b);   // 1  (나머지)
```

### 비트 연산자 (임베디드 필수)

레지스터 제어에 자주 사용합니다:

```c
uint8_t reg = 0b00000000;

reg |= (1 << 3);   // 3번 비트 SET   → 0b00001000
reg &= ~(1 << 3);  // 3번 비트 CLEAR → 0b00000000
reg ^= (1 << 3);   // 3번 비트 TOGGLE
reg >>= 2;         // 오른쪽으로 2비트 이동 (÷4)
reg <<= 1;         // 왼쪽으로 1비트 이동 (×2)

// 특정 비트 확인
if (reg & (1 << 3)) {
    // 3번 비트가 1인 경우
}
```

### 비교 / 논리 연산자

```c
int a = 5, b = 10;

// 비교
a == b   // false (0)
a != b   // true  (1)
a < b    // true
a >= b   // false

// 논리
(a > 0) && (b > 0)  // AND: 둘 다 참 → true
(a > 0) || (b < 0)  // OR:  하나라도 참 → true
!(a == b)           // NOT: false → true
```

## printf 형식 지정자

```c
printf("%d",  42);          // 정수
printf("%u",  42u);         // 부호 없는 정수
printf("%f",  3.14);        // 실수
printf("%.2f", 3.14159);    // 소수점 2자리
printf("%s",  "hello");     // 문자열
printf("%c",  'A');         // 문자
printf("%x",  255);         // 16진수: ff
printf("%08x", 255);        // 16진수 8자리 패딩: 000000ff
printf("%p",  &a);          // 포인터 주소
```

## 정리

- 임베디드에서는 `stdint.h`의 고정 크기 타입을 사용
- 비트 연산자(`|`, `&`, `^`, `~`, `<<`, `>>`)는 레지스터 제어의 핵심
- `printf` 형식 지정자를 정확히 맞추지 않으면 미정의 동작 발생
