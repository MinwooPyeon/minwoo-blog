## 함수 기본 구조

```c
반환타입 함수이름(매개변수 목록) {
    // 함수 본문
    return 반환값;
}
```

```c
// 두 정수의 최댓값 반환
int max(int a, int b) {
    return (a > b) ? a : b;
}

int main(void) {
    int result = max(10, 20);
    printf("최댓값: %d\n", result);  // 20
    return 0;
}
```

## 함수 프로토타입 (선언)

사용 전 선언이 필요합니다. 헤더 파일에 프로토타입을 배치하는 것이 일반적입니다:

```c
// sensor.h
#ifndef SENSOR_H
#define SENSOR_H

float read_temperature(void);
int   read_humidity(void);
void  calibrate_sensor(float offset);

#endif
```

```c
// sensor.c
#include "sensor.h"

float read_temperature(void) {
    // 하드웨어 레지스터에서 값 읽기
    return 25.3f;
}
```

## 매개변수 전달 방식

### 값에 의한 전달 (Call by Value)

```c
void increment(int x) {
    x++;  // 원본에 영향 없음
}

int n = 5;
increment(n);
printf("%d\n", n);  // 여전히 5
```

### 포인터에 의한 전달 (Call by Reference)

```c
void increment(int *x) {
    (*x)++;  // 원본 값 변경
}

int n = 5;
increment(&n);
printf("%d\n", n);  // 6
```

### 배열 전달

배열은 포인터로 전달됩니다:

```c
float average(int *arr, int size) {
    int sum = 0;
    for (int i = 0; i < size; i++) {
        sum += arr[i];
    }
    return (float)sum / size;
}

int data[] = {10, 20, 30, 40, 50};
float avg = average(data, 5);
printf("평균: %.1f\n", avg);  // 30.0
```

## void 함수와 반환값 없는 함수

```c
// 반환값 없음
void blink_led(int pin, int count) {
    for (int i = 0; i < count; i++) {
        gpio_write(pin, HIGH);
        delay_ms(500);
        gpio_write(pin, LOW);
        delay_ms(500);
    }
}
```

## 재귀 함수

```c
// 팩토리얼
int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}

// 피보나치
int fibonacci(int n) {
    if (n <= 1) return n;
    return fibonacci(n - 1) + fibonacci(n - 2);
}
```

> 임베디드에서 재귀는 스택 오버플로 위험이 있습니다. 깊이가 깊어질 수 있다면 반복문으로 대체하세요.

## static 함수와 변수

```c
// 파일 내부에서만 사용하는 함수 (다른 파일에서 접근 불가)
static void internal_helper(void) {
    // ...
}

// static 지역변수: 함수 종료 후에도 값 유지
int call_count(void) {
    static int count = 0;  // 최초 1회만 초기화
    return ++count;
}

printf("%d\n", call_count());  // 1
printf("%d\n", call_count());  // 2
printf("%d\n", call_count());  // 3
```

## inline 함수

짧고 자주 호출되는 함수에 사용해 함수 호출 오버헤드를 줄입니다:

```c
static inline uint8_t clamp_byte(int val) {
    if (val < 0)   return 0;
    if (val > 255) return 255;
    return (uint8_t)val;
}
```
