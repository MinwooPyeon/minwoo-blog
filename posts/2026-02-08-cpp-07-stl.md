## vector

동적 크기 배열. C++에서 가장 많이 쓰는 컨테이너:

```cpp
#include <vector>

std::vector<int> v = {3, 1, 4, 1, 5};

v.push_back(9);      // 뒤에 추가
v.pop_back();        // 뒤에서 제거
v.insert(v.begin() + 2, 99);  // 인덱스 2에 삽입
v.erase(v.begin() + 2);       // 인덱스 2 제거

std::cout << v.size()     << std::endl;  // 크기
std::cout << v.capacity() << std::endl;  // 할당된 용량
v.reserve(100);  // 미리 공간 확보 (재할당 방지)

// 순회
for (const auto &x : v) std::cout << x << " ";
for (size_t i = 0; i < v.size(); i++) std::cout << v[i] << " ";
```

## array (고정 크기)

임베디드에서 동적 할당 없이 사용:

```cpp
#include <array>

std::array<float, 5> readings = {1.1f, 2.2f, 3.3f, 4.4f, 5.5f};

std::cout << readings.size()  << std::endl;  // 5
std::cout << readings.front() << std::endl;  // 1.1
std::cout << readings.back()  << std::endl;  // 5.5

// 컴파일 타임에 크기 확정 → 스택 할당
```

## map / unordered_map

키-값 쌍 저장:

```cpp
#include <map>
#include <unordered_map>

// map: 키 정렬, O(log n) 탐색
std::map<std::string, float> sensor_map;
sensor_map["temperature"] = 25.3f;
sensor_map["humidity"]    = 60.0f;
sensor_map["pressure"]    = 1013.25f;

for (const auto &[key, val] : sensor_map) {
    std::cout << key << ": " << val << std::endl;
}

// unordered_map: 정렬 없음, O(1) 탐색
std::unordered_map<int, std::string> id_map;
id_map[1] = "sensor_a";
id_map[2] = "sensor_b";

if (id_map.count(1)) {
    std::cout << id_map[1] << std::endl;
}
```

## set / queue / stack

```cpp
#include <set>
#include <queue>
#include <stack>

// set: 중복 없는 정렬 집합
std::set<int> ids = {3, 1, 4, 1, 5};  // {1, 3, 4, 5}

// queue: FIFO (메시지 큐)
std::queue<std::string> msg_queue;
msg_queue.push("cmd_start");
msg_queue.push("cmd_stop");
std::cout << msg_queue.front() << std::endl;  // cmd_start
msg_queue.pop();

// stack: LIFO
std::stack<int> stk;
stk.push(1); stk.push(2); stk.push(3);
std::cout << stk.top() << std::endl;  // 3
stk.pop();
```

## algorithm

```cpp
#include <algorithm>
#include <numeric>

std::vector<int> v = {5, 3, 8, 1, 9, 2};

// 정렬
std::sort(v.begin(), v.end());           // 오름차순
std::sort(v.begin(), v.end(), std::greater<int>());  // 내림차순

// 탐색
auto it = std::find(v.begin(), v.end(), 8);
if (it != v.end()) std::cout << "찾음: " << *it << std::endl;

// 최솟값 / 최댓값
auto [mn, mx] = std::minmax_element(v.begin(), v.end());
std::cout << "min=" << *mn << " max=" << *mx << std::endl;

// 합계 / 평균
int total = std::accumulate(v.begin(), v.end(), 0);
double avg = static_cast<double>(total) / v.size();

// 변환 (map)
std::vector<float> scaled;
std::transform(v.begin(), v.end(), std::back_inserter(scaled),
               [](int x) { return x * 0.1f; });

// 필터 (remove_if + erase)
v.erase(std::remove_if(v.begin(), v.end(),
        [](int x) { return x % 2 == 0; }),  // 짝수 제거
        v.end());
```

## string

```cpp
#include <string>

std::string s = "Hello, Embedded!";

s.length();           // 16
s.substr(7, 8);       // "Embedded"
s.find("Embedded");   // 7 (위치)
s.replace(7, 8, "World");  // "Hello, World!"

// 숫자 ↔ 문자열
std::string str = std::to_string(25.3f);
float val = std::stof("3.14");
int   num = std::stoi("42");

// 분리
std::istringstream ss("10,20,30");
std::string token;
while (std::getline(ss, token, ',')) {
    std::cout << std::stoi(token) << std::endl;
}
```

## 정리

- `vector`: 일반 동적 배열, `array`: 고정 크기 배열 (임베디드 선호)
- `map`: 정렬된 키-값, `unordered_map`: 빠른 해시 탐색
- `algorithm`으로 정렬, 탐색, 변환을 한 줄로
- 람다(`[](){}`)와 STL 알고리즘 조합이 현대 C++ 스타일
