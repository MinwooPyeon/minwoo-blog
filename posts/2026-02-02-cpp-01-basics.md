## C와 C++의 주요 차이점

| 항목 | C | C++ |
|---|---|---|
| 패러다임 | 절차적 | 절차적 + 객체지향 + 제네릭 |
| 입출력 | `printf` / `scanf` | `cout` / `cin` |
| 문자열 | `char[]` | `std::string` |
| 메모리 | `malloc` / `free` | `new` / `delete` |
| 타입 캐스팅 | `(type)` | `static_cast<type>()` |
| 주석 | `/* */` | `/* */` + `//` |

## 기본 입출력

```cpp
#include <iostream>
using namespace std;

int main() {
    int x;
    cout << "숫자 입력: ";
    cin >> x;
    cout << "입력값: " << x << endl;
    return 0;
}
```

### cout 포맷팅


## 참조자 (Reference)

C++의 핵심 기능입니다. 포인터보다 안전하고 편리합니다:

```cpp
int  x = 10;
int &ref = x;   // ref는 x의 별칭 (별도 메모리 없음)

ref = 20;
cout << x << endl;  // 20

// 참조자로 swap (포인터 없이)
void swap(int &a, int &b) {
    int tmp = a;
    a = b;
    b = tmp;
}

int a = 3, b = 7;
swap(a, b);
cout << a << " " << b << endl;  // 7 3
```

### const 참조자 (함수 매개변수 최적화)

```cpp
// 복사 비용 없이 읽기 전용으로 전달
void print_info(const string &name, const vector<int> &data) {
    cout << name << ": ";
    for (int v : data) cout << v << " ";
    cout << endl;
}
```

## auto 키워드

타입 추론으로 코드를 간결하게:

```cpp
auto x = 42;          // int
auto y = 3.14;        // double
auto s = "hello"s;    // std::string

// 범위 기반 for
vector<int> nums = {1, 2, 3, 4, 5};
for (auto n : nums) {
    cout << n << " ";
}

// const 참조로 순회 (대형 객체 복사 방지)
for (const auto &n : nums) {
    cout << n << " ";
}
```

## nullptr

C++11 이후 `NULL` 대신 사용합니다:

```cpp
int *p = nullptr;   // 타입 안전한 null 포인터

if (p == nullptr) {
    cout << "포인터가 비어있음\n";
}

// C의 NULL은 0(int)이지만 nullptr는 포인터 타입
void func(int x)    { cout << "int\n"; }
void func(int *p)   { cout << "ptr\n"; }

func(NULL);     // 모호 (컴파일러 경고)
func(nullptr);  // "ptr" 명확
```

## new / delete

```cpp
// 단일 객체
int *p = new int(42);
cout << *p << endl;  // 42
delete p;
p = nullptr;

// 배열
int *arr = new int[10]{0};  // 0으로 초기화
arr[0] = 1;
delete[] arr;  // [] 필수!
arr = nullptr;
```

> C++11 이후로는 `new`/`delete` 대신 스마트 포인터(`unique_ptr`, `shared_ptr`)를 사용합니다.

## namespace

이름 충돌 방지:

```cpp
namespace sensor {
    void init() { cout << "sensor init\n"; }
    int read()  { return 100; }
}

namespace motor {
    void init() { cout << "motor init\n"; }
}

sensor::init();
motor::init();

// using으로 생략
using namespace sensor;
init();  // sensor::init() 호출
```

## 정리

- 참조자(`&`)는 포인터보다 안전하고 직관적인 별칭
- `auto`로 타입 추론, 범위 기반 for로 순회 간결화
- `nullptr`는 `NULL`보다 타입 안전
- `new`/`delete` 사용 후에는 포인터를 `nullptr`로 초기화
