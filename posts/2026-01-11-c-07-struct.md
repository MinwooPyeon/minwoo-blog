## 구조체 (struct)

연관된 데이터를 하나로 묶습니다:

```c
struct Sensor {
    char  name[20];
    float temperature;
    float humidity;
    int   is_valid;
};

struct Sensor s;
strcpy(s.name, "DHT22");
s.temperature = 25.3f;
s.humidity    = 60.0f;
s.is_valid    = 1;

printf("[%s] 온도: %.1f, 습도: %.1f\n",
        s.name, s.temperature, s.humidity);
```

### typedef로 간결하게

```c
typedef struct {
    float x;
    float y;
    float z;
} Vector3;

Vector3 accel = {0.1f, 0.0f, -9.8f};
printf("가속도: (%.2f, %.2f, %.2f)\n", accel.x, accel.y, accel.z);
```

## 구조체 포인터

```c
typedef struct {
    int   id;
    float value;
} SensorData;

SensorData data = {1, 23.5f};
SensorData *p   = &data;

// . 대신 -> 사용
printf("ID: %d, 값: %.1f\n", p->id, p->value);

// 아래 두 표현은 동일
p->id   = 2;
(*p).id = 2;
```

### 구조체 배열

```c
SensorData sensors[3] = {
    {1, 12.5f},
    {2, 25.0f},
    {3, 37.5f}
};

for (int i = 0; i < 3; i++) {
    printf("센서 %d: %.1f\n", sensors[i].id, sensors[i].value);
}
```

## 중첩 구조체

```c
typedef struct {
    int year;
    int month;
    int day;
} Date;

typedef struct {
    char name[30];
    Date birthday;
    float gpa;

Student s = {"편민우", {2001, 6, 4}, 4.2f};
printf("%s — %d년 %d월 %d일\n",
        s.name, s.birthday.year, s.birthday.month, s.birthday.day);
```

## 구조체 패딩

컴파일러는 정렬을 위해 자동으로 패딩을 삽입합니다:

```c
struct A {
    char  c;    // 1 byte
    // 3 byte padding
    int   i;    // 4 byte
    char  c2;   // 1 byte
    // 3 byte padding
};
// sizeof(struct A) = 12, 예상과 다를 수 있음

// 패딩 없애기 (통신 패킷, 파일 포맷에 사용)
struct __attribute__((packed)) Packet {
    uint8_t  header;
    uint32_t data;
    uint8_t  checksum;
};
// sizeof == 6
```

## 공용체 (union)

같은 메모리를 여러 타입으로 해석합니다:

```c
typedef union {
    uint32_t raw;
    struct {
        uint8_t b0;
        uint8_t b1;
        uint8_t b2;
        uint8_t b3;
    } bytes;
    float f;
} Data32;

Data32 d;
d.f = 3.14f;
printf("float: %f\n", d.f);
printf("raw:   0x%08X\n", d.raw);
printf("bytes: %02X %02X %02X %02X\n",
        d.bytes.b0, d.bytes.b1, d.bytes.b2, d.bytes.b3);
```

## 비트 필드

구조체 멤버를 비트 단위로 선언합니다. 레지스터 맵 표현에 유용합니다:

```c
typedef struct {
    uint8_t enable   : 1;  // 1비트
    uint8_t mode     : 2;  // 2비트 (0~3)
    uint8_t speed    : 3;  // 3비트 (0~7)
    uint8_t reserved : 2;  // 예약 비트
} ControlReg;

ControlReg ctrl = {0};
ctrl.enable = 1;
ctrl.mode   = 2;
ctrl.speed  = 5;
```

## 정리

- `struct`로 관련 데이터를 묶어 가독성과 유지보수성 향상
- `->` 연산자로 포인터를 통한 멤버 접근
- `__attribute__((packed))`로 패킷 구조체에서 패딩 제거
- `union`으로 같은 메모리를 다른 타입으로 해석 (엔디안 변환 등)
- 비트 필드로 레지스터 맵을 직관적으로 표현
